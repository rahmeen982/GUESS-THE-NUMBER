# Number Guessing Game

A simple number guessing game built using HTML, CSS, JavaScript, and jQuery.

 ## How It Works

- The game generates a random number between 1 and 100.
- The player has 5 attempts to guess the correct number.
- The game provides hints:
  - 📈 Too High
  - 📉 Too Low
- If the player guesses correctly, they win.
- If attempts run out, the game ends.

## Technologies Used

- HTML
- CSS
- JavaScript
- jQuery

##  Features

- Random number generation
- Attempt counter
- Restart button
- User feedback messages
